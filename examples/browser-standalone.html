Browser Standalone Example

ℹ️ Info: This page demonstrates using the UMD build directly in the browser. The iframe resizing library is loaded via a script tag and available globally.

Current Height: 0px

This content will automatically notify the parent window about height changes.

How to Use

Include the UMD build in your HTML:

<script src="path/to/iframe-resizing.umd.js"></script> <script> // Option 1: Auto-init (recommended) const cleanup = IFrameResizing.autoInit(); // Option 2: Manual init // const cleanup = IFrameResizing.init({ // onError: (error) => console.error(error) // }); </script>

CDN Usage (after npm publish)

You can also load it from a CDN:

<!-- From unpkg --> <script src="https://unpkg.com/@andsafe/iframe-resizing@1.0.0/dist/iframe-resizing.umd.js"></script> <!-- From jsDelivr --> <script src="https://cdn.jsdelivr.net/npm/@andsafe/iframe-resizing@1.0.0/dist/iframe-resizing.umd.js"></script>
